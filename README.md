Meesho Clone (HTML & CSS)

A responsive **Meesho homepage clone** built using **HTML and CSS**.

Project Overview
This project recreates the **Meesho homepage banner and header layout** to practice concepts such as:
* Clean and semantic HTML structure
* Reusable CSS components
* Navigation bar and menu design
* Banner layout with background images
* Flexbox-based alignment
* Responsive design using media queries

Technologies Used
HTML5 – Page structure
CSS – Styling and layout
Flexbox – Alignment and layout control
Media Queries – Mobile responsiveness

Pages Included
Home Page (index.html)
* Top header with logo, search bar, and navigation links
* Horizontal category navigation menu
* Large promotional banner section
* Call-to-action button (Shop Now)
* Information strip highlighting platform benefits
* Clean Meesho-inspired UI layout

UI Features
* Light e-commerce style color theme
* Clean header and navigation design
* Large banner with background image
* Rounded info strip layout
* Reusable text and button styles
* Proper spacing and typography hierarchy

Responsive Design:css
The layout adapts to different screen sizes using CSS media queries:
* Header elements wrap properly on small screens
* Horizontal menu becomes scrollable on mobile
* Banner text scales down for smaller devices
* Info section stacks vertically on mobile

what do we learn
* Understanding real-world e-commerce UI layouts
* Practicing Flexbox and layout positioning
* Designing responsive navigation bars
* Structuring banner-based landing sections
* Improving CSS organization and reusability
